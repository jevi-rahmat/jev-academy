# Jev-Academy
Jev-Academy or Jev-Glossary is a LMS website project to provide online course built using PHP, MySQL, JavaScript, and TailwindCSS.

### Features
- Authentication
User who is not yet logged in can view and browse the available courses but to be able to enroll, user must log in first. Register, log in, and log out features are functional.

- Browse Course
User can view courses and preview its content. User can also enroll by purchasing or using code to a course.

- Create Course
User registered as Instructor can create their own course. The material of the course is in markdown format `.md`. Course is structured into:
  - Sections: The theme for a module.
  - Contents: The material of the module.
 
- Catalog
Intended to support the following major:
  - Mechanical Engineering
  - Civil Engineering
  - Electrical Engineering
  - Information Technology
  - Business
  - Accounting
  - English Language
 
  However, the field can be as specific as possible. Such as, "Thermodynamic I", "Calculus I", etc.

- Account
Implemented profile dashboard to monitor user's course or enrolled course.

- Monetary
Implemented monetary features to simulate transactions using dummy data.

### Role

- Student
Student can browse and enroll into a course, top-up, and made purchases.

- Instructor
Instructor can broswe, create, update, and delete their created course

- Administrator
Adminstrator can browser, manage users, manage courses, manage monetary-related features.
